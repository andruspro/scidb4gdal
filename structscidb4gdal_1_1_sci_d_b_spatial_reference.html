Detailed Description

A carrier structure for holding information about the spatial reference.

A structure for storing spatial reference that is to be obtained from either the source file or from the SciDB database. This reference is restricted to a two dimensional projection with the dimension x and y. X can be thought of the west-east direction and Y is the north-south direction.
