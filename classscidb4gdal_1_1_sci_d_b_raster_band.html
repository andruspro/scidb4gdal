GDAL function called as array attribute data is requested, loads data from SciDB server.

This function loads a block data from the server and it stores it temporally in a Tile cache if needed, before the data is written into the stated image.

Parameters
nBlockXOffthe column offset as a number
nBlockYOffthe row offset as a number
